Build a complete production-ready full-stack Smart Parking Booking Web Application using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

The website must have a modern premium UI, responsive design, secure authentication system, real-time parking management features, QR-based booking verification, payment integration, and role-based dashboards.

The project architecture must follow industry standards with clean folder structure, reusable components, protected routes, REST APIs, JWT authentication, middleware, MVC architecture, error handling, validation, and scalable database design.

# PARKING GUIDE FEATURES

Parking Guide dashboard must allow:

## Slot Monitoring

Guide can:

* View all slots in assigned parking area
* View available slots
* View occupied slots
* View reserved slots
* View expired bookings

---

## Booking Verification

When user books a slot:

* Generate QR Code
* Guide scans QR code
* Guide sees booking details

QR Details:

* User Name
* Vehicle Number
* Slot ID
* Parking Area
* Booking Time
* Reserved Hours
* Estimated Amount
* Booking Status

---

## Check-In System

Guide can:

* Confirm user arrival
* Set actual check-in time
* Mark slot as occupied

Rules:

* If user does not check in before reservation expiry time:

  * Booking automatically expires
  * Slot becomes available again

---

## Check-Out System

Guide can:

* Set check-out time
* Calculate extra parking charges if delayed
* Generate final bill
* Free the slot after checkout

Billing Logic:

* Base price calculated from reserved hours
* Additional hourly charge added for overtime

Example:
Reserved: 2 hours
Stayed: 3.5 hours
Extra amount automatically calculated

## Slot Booking System

Booking flow:

1. Select parking area
2. Select available slot
3. Enter vehicle details
4. Select number of hours
5. Make payment
6. Receive booking confirmation
7. Receive QR Code

Vehicle details:

* Vehicle Number
* Owner Name

---

## QR Code System

After successful booking:

* Generate downloadable QR Code
* Show booking details
* Send QR code via email

QR should contain:

* Booking ID
* User ID
* Slot ID
* Area ID
* Reserved Time
* Payment Status

---

## Booking Rules

If user:

* Does not check in before expiry:

  * Booking cancelled automatically
  * Slot released

If user:

* Checks out late:

  * Additional amount calculated automatically
